# Kotlin Incident Response Tracker

## Overview
This project is a Kotlin console application for managing cybersecurity incidents. It allows a user to create incidents, assign severity levels, update incident status, assign analysts, add evidence notes, filter incidents, and display a summary dashboard.

## Purpose
The purpose of this software is to practice Kotlin programming concepts in a project that is both practical and engaging. This project demonstrates how Kotlin can be used to organize structured data and manage a workflow similar to a real-world incident response process.

## Features
- Create new incidents
- Assign a severity level
- Assign an analyst to an incident
- View all incidents
- View detailed information for a specific incident
- Add evidence notes to incidents
- Update the incident status
- Reassign incidents to other analysts
- Filter incidents by severity
- Display a dashboard summary
